# 주간보고서 양식

# 주간보고서 저장
"내문서" - "주간보고" 폴더에 생성한 주간보고서 파일 저장함.

# 주간보고서 발송
지정한 보고서 담당자(성명, email주소)에게 주간보고서 발송. 메일 본문에 간단하게 요약해서 작성함.

# Human in the loop
주간보고서 생성이 완료되면, 사용자에게 컨펌 요청. 사용자가 컨펌하면 발송. 
컨펌하지 않고 수정사항을 요청하면 수정하여 다시 생성. (ppt파일에 날짜와 version을 파일명에 기입)
